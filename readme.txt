WooCommerce Mobile Checkout with Table display all WooCommerce products in Table View. 

== Description ==

Mobile Checkout with Table (MCT) is WooCommerce plugin display all products in table with sorting, searching and filter by categories. Simple and fast UI allow clients to see all products on one page. Almost all the fields are rendered in each columns. 

= Features =
* All Products in Table View
* Search Products
* Sorting Products
* Pagination
* Image Thumbs
* Responsive 
* Updated DataTable API
* Filters & Actions

== FAQs ==
= Does it replace default Shop page? =
* No it does not replace default shop page
= How to use this? =
* Create a page and paste this shortcode: [wmct]
= Product Columns Show/Hide? =
* In free version, five basic columns are shown, please see screenshot for more detail

= PRO Version =
* Filter by Categories
* Show/hide columns
* Grid View
* Generate Shortcode for Selected Categories

== Installation ==

1. Upload `woo-mct` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once Plugin is activated, `Woo MCT Settings` menu page will appear in Dashboard
